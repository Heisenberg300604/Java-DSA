# ☕ Java DSA — Learning Repository

> A public learning journal documenting my journey through Java and Data Structures & Algorithms — from the very basics to solving problems on LeetCode.

## 📖 About

This repository is my personal Java + DSA learning space. As I work through concepts, I push all my code here — organized by topic — so it's easy to revisit, reference, and share with anyone on the same learning path.

Whether you're just getting started with Java or grinding DSA problems, feel free to explore, learn, and use any code here.

1. Clone the repository:

2. Open in your preferred IDE (IntelliJ IDEA recommended).
3. Navigate to any topic folder and run the `.java` files directly.
